Install TexLive in your OS.

Install latex2pdf which is dependent on TexLive. 

cpan App::cpanminus # handy tool to install cpan module
cpanm Template # Template is missing on cygwin
cpanm latex2pdf # finally

Convert now! 

pdf2latex cv.tex
