    Tags :

Please note that data import could be very slow in this way. 

mysqldump -u root -p111111 -c --compact -t --skip-opt dbname tbl1 tbl2|sed -e "s/\\\'/''/g"|gzip -9 > data.dump.gz
zcat data.dump.gz | sqlite3 db.sqlite3
